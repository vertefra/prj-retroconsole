# Module 1: The Linux Workspace

This module covers setting up your professional CAD environment on Linux.

## Step 1: Installation & Launch
- **Action:** Download the **FreeCAD 0.21.2 AppImage** (or the RealThunder branch for experts) from [FreeCAD.org](https://www.freecad.org/downloads.php).
- **Action:** Make the file executable: `chmod +x FreeCAD_xxx.AppImage`.
- **Action:** Launch the application.
- **Check:** Ensure the "Start Page" loads correctly without any library errors in the report view.
- **Goal:** Get the CAD environment running.

## Step 2: Set Navigation Style
- **Action:** Go to `Edit -> Preferences -> Display -> Navigation`.
- **Action:** Set the navigation style to **Blender** (highly recommended for drawing tablets) or **CAD**.
- **Check/Goal:** You should be able to create a dummy document (`Ctrl+N`), create a box in the `Part` workbench, and practice rotating it (using your pen button or middle mouse button).

## Step 3: Workbench Familiarization
- **Action:** Locate the Workbench selector at the top center.
- **Action:** Switch between `Section`, `PartDesign`, and `Sketcher` to see how the toolbars change.
- **Goal:** Understand where the specialized tools live.
