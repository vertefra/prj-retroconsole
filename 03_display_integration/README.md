# Module 3: Display Integration

Modeling the cutout and mount for the HMTECH 7" screen.

## Step 1: Screen Dimensions
- **Action:** Open your `Properties` spreadsheet.
- **Action:** Add `Screen_W = 165` and `Screen_H = 104`.
- **Goal:** Centralize display dimensions.

## Step 2: Creating the Bezel
- **Action:** Create a new `Body` for the screen frame.
- **Action:** Sketch the negative space (the hole the screen will fit through).
- **Action:** Add a 1mm offset for "Tolerance" so the screen isn't too tight.
- **Goal:** Design the primary viewing window.
