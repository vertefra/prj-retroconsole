# Module 5: Enclosure Design

Finalizing the arci shell.

## Step 1: Shelling
- **Action:** Use the `Thickness` tool in `PartDesign` to hollow out your main solid.
- **Action:** Set the wall thickness to 3mm (ideal for 3D printing).
- **Goal:** Create internal volume for electronics.

## Step 2: Port Cutouts
- **Action:** Reference `component_specs.md` to create the USB and Power cutouts.
- **Goal:** External connectivity.
