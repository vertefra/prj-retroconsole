# arci Interactive FreeCAD Course

This tutorial is organized into modules. Each folder contains a detailed `README.md` for that step.

## 📚 Course Modules

1.  **[Module 1: Workspace Setup](./01_workspace_setup/README.md)**
    - Install FreeCAD and configure navigation.
2.  **[Module 2: Pi 4 Modeling](./02_pi4_modeling/README.md)**

3.  **[Module 3: Display Integration](./03_display_integration/README.md)**
    - Design the screen bezel and mounting.
4.  **[Module 4: Cooling & Controls](./04_cooling_and_controls/README.md)**
    - Add the iUniker fan and Arcade buttons.
5.  **[Module 5: Enclosure Design](./05_enclosure_design/README.md)**
    - Hollow out the shell and add port cutouts.
6.  **[Module 6: Manufacturing Prep](./06_manufacturing_prep/README.md)**
    - Final inspection and export for 3D printing.
